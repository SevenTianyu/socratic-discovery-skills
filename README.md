# Socratic Discovery Skills

This repository contains two language-specific Codex skills for guided Socratic exploration:

- `socratic-discovery-zh`: Chinese-first version of "Cyber Socrates"
- `socratic-discovery-en`: English-first version of "Cyber Socrates"

Both skills are designed to help a user explore confusion, blind spots, hidden assumptions, and "unknown unknowns" through one question at a time instead of jumping straight to answers.

## Skills

- [socratic-discovery-zh/SKILL.md](./socratic-discovery-zh/SKILL.md)
- [socratic-discovery-en/SKILL.md](./socratic-discovery-en/SKILL.md)

## Install

Copy either skill folder, or both, into your Codex skills directory:

- Windows: `C:\Users\<you>\.codex\skills`
- macOS/Linux: `~/.codex/skills`

## Invoke

- `Use $socratic-discovery-zh ...`
- `Use $socratic-discovery-en ...`
